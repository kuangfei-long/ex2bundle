# ACL 2020 dataset

This dataset is related to this paper: [Extractive Summarization as Text Matching](https://arxiv.org/abs/2004.08795)

Please find the train/test/validate dataset and the generated summary in the github repo [here](https://github.com/maszhongming/MatchSum?tab=readme-ov-file)

If want to use the dataset, follow the steps:

1. download the dataset from the link in github
2. run the script: extra_data/GPT_ACL.py (may need to changed the path setting)
